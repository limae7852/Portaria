Como configurar
  1. Crie um projeto no Firebase Console (https://console.firebase.google.com).
  2. Ative o Firestore (modo de produção ou teste) em Cloud Firestore.
  3. Copie as credenciais do seu app (objeto firebaseConfig).
  4. Cole as credenciais no arquivo HTML dentro do bloco /* --- CONFIGURE AQUI --- */.
  5. Sirva o arquivo localmente (ex: Live Server) ou publique em um host estático.
Observação: este exemplo usa Firestore. Se preferir Realtime Database ou Firebase Auth, eu adapto.
